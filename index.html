A full-stack expense-splitting web app supporting group creation, shared expense tracking, and automatic equal/custom-split cost calculation across multiple users. Includes a debt-simplification algorithm that reduces multi-party balances to a minimal set of settle-up transactions — replacing O(n²) pairwise debts with a one-click settle-up flow.
